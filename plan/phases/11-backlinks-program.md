## 5. Data report trimestriel linkable — Cahier des charges

### Titre : "Baromètre des frais réels des PSAN agréés — Q2 2026"

### Pourquoi c'est linkable
Le sujet "frais cachés crypto" est récurrent dans la presse FR sans data primaire indépendante. Cryptoreflex devient la **seule source citable** avec méthodologie publique.

### Structure (40 pages PDF + landing HTML)
1. **Executive summary** (1 page) : 5 chiffres-chocs (ex : "1,42% écart médian entre frais affichés et frais réels")
2. **Méthodologie** (3 pages) : protocole d'audit, jeu de tests (10 transactions standardisées par PSAN), période d'observation, sources spread/slippage, code open-source GitHub
3. **Panel** (2 pages) : 18 PSAN agréés AMF au T2 2026 (liste exhaustive)
4. **Résultats détaillés** (15 pages) : fiche par PSAN — fees affichés vs réels, slippage médian, écart spread, frais cachés (withdrawals, conversion, gas, etc.)
5. **Classement transparent** (4 pages) : top 5 par catégorie (débutant / actif / DCA / staking / large cap only)
6. **Évolution YoY** (3 pages) : comparaison Q2 2025 / Q2 2026 si data disponible (sinon baseline)
7. **Focus MiCA** (5 pages) : impact des nouvelles obligations RTS sur la transparence
8. **Recommandations consommateur** (3 pages)
9. **Annexes méthodologiques** (4 pages)

### Assets de diffusion (linkable)
- PDF téléchargeable contre email (lead magnet)
- Landing HTML interactive avec graphes filtrables (Recharts)
- 1 dataset CSV public sur GitHub (citable académique)
- 6 visuels embeddables (`<iframe>` + `<img>` d'incorporation) — chaque embed = backlink
- Communiqué de presse (template section 6)
- Tweet thread / LinkedIn post / Reddit r/CryptoFrance

### Calendrier
- T+0 : recueil données (semaines 1-3)
- T+3 : rédaction & design (semaines 4-6)
- T+6 : pre-pitch sous embargo à 8 journalistes tier-1 (J-7)
- T+7 : publication + diffusion (J0 release)
- T+8 à T+12 : campagne d'outreach (4 semaines)

### KPI / report
- 25 backlinks par édition (donc 100 backlinks/an si 4 reports)
- 1 mention presse tier-1 minimum (Capital, Echos, BFM)
- 500 téléchargements PDF
- DR moyen des liens : 50

### Coût
- Design : 800 EUR (freelance Figma)
- Dev landing : interne (template Next.js réutilisable)
- Diffusion presse : 250 EUR (Cision ciblé) ou gratuit via outreach manuel

## 14. Risques & garde-fous

- **Pénalité Google manual action** : éviter PBN, échanges 1-pour-1 mécaniques, paid links non disclosed. Respecter Google Webmaster Guidelines.
- **Anchor over-optimization** : capper anchor "comparateur crypto" à 8% du profil global.
- **Wikipedia revert** : risque modéré, traité section 8 (transparence COI).
- **Saisonnalité** : pic presse crypto = janvier (déclaration fiscale) + cycles bull market. Caler Baromètre Q1 sur déclaration fiscale (publication mi-février).
- **Concurrence** : si Coinhouse, Bitpanda, Kraken FR lancent leur propre baromètre, garder le différenciant méthodologie open-source.
